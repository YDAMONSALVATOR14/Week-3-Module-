# 🚀 Week 3 Module – Advanced Sorting Algorithms in Java

Welcome to my **Week 3 Data Structures & Algorithms (DSA)** repository! This project contains Java implementations of fundamental sorting algorithms covered during my academic coursework at **KCC Institute of Technology & Management**.

## 📖 About

This repository demonstrates the implementation and working of the following sorting algorithms:

- 🔀 Merge Sort
- ⚡ Quick Sort
- 🌳 Heap Sort

Each program is written in **Java** with a focus on code readability, efficiency, and understanding the underlying concepts.

## ▶️ How to Run

Compile the program:

```bash

```

Run the program:

```bash
java FileName
```

**Example:**

```bash
javac MergeSort.java
java MergeSort
```

## 📊 Time Complexity

| Algorithm | Best | Average | Worst |
|-----------|:----:|:-------:|:-----:|
| Merge Sort | O(n log n) | O(n log n) | O(n log n) |
| Quick Sort | O(n log n) | O(n log n) | O(n²) |
| Heap Sort | O(n log n) | O(n log n) | O(n log n) |
