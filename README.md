# 🌟 Algorithms Included

## 🔀 Merge Sort

Efficiently sorts data using the **Divide and Conquer** strategy by recursively splitting arrays into smaller subarrays and merging them back in sorted order.

**Concept Used:** Divide & Conquer

---

## ⚡ Quick Sort

Sorts elements by selecting a pivot and partitioning the array, making it one of the fastest sorting algorithms for average-case performance.

**Concept Used:** Divide & Conquer & Partitioning

---

## 🌳 Heap Sort

Builds a Max Heap and repeatedly extracts the largest element to produce a sorted array with guaranteed **O(n log n)** complexity.

**Concept Used:** Heap Data Structure
